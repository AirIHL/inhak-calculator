calculator2

클래스를 적용해 기본적인 연산을 수행할 수 있는 계산기 만들기

간단한 콘솔 기반 계산기 애플리케이션으로, 사용자 입력을 통해 기본적인 사칙연산을 수행합니다.

주요 기능
    기본 연산 : 덧셈, 뺄셈, 곱셈, 나눗셈

    사용자 인터페이스 : 사용자로부터 두 개의 숫자와 연산자를 입력 받음
                     연속 계산 기능 지원
                     'exit' 입력으로 프로그램 종료

    연산 결과를 저장 : LinkedList에 연산 결과를 저장함(최대 6개)
                     저장된 값을 보여줌

    저장된 데이터 삭제 : LinkedList에 연산 결과를 저장한 것이 6개를 초과하면 가장 먼저 저장된 데이터를 삭제함


구현 사항
    입력 처리
    Scanner 클래스를 사용하여 사용자 입력을 받음
    첫 번째 숫자, 연산자, 두 번째 숫자 순으로 입력

    연산 처리
    switch 문을 사용하여 연산자에 따른 계산 수행
    잘못된 연산자 입력 시 오류 메시지 출력

    캡슐화
    결과 데이터는 오직 getter/setter 메서드를 통해서만 접근
    getResultData(): 결과값을 조회
    setRemoveFirstResult(): 첫 번째 결과를 제거
    계산 결과를 private LinkedList인 resultData에 저장

while 루프를 사용한 연속 계산 지원하며
사용자가 'exit'를 입력할 때까지 계산을 반복합니다.

사용 방법
    1. 첫 번째 숫자 입력
    2. 사칙연산 기호 입력 (+, -, *, /)
    3. 두 번째 숫자 입력
    4. 계산 결과 확인
    5. 계속하려면 아무 키나 입력, 종료하려면 'exit' 입력

제한사항
    정수 연산만 지원
    기본적인 사칙연산만 가능
    예외 처리가 구현되지 않음 (예: 0으로 나누기, 숫자가 아닌 다른 값 넣기)
